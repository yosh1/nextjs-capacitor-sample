# Next.js with Capacitor

## add package

```
$ yarn add @capacitor/core @capacitor/cli
```

## build app with capacitor

```
$ npx cap init --web-dir=out   
```

## add platform

```
$ yarn add @capacitor/ios @capacitor/android
```

and

```
$ npx cap add ios
$ npx cap add android
```

## opening & running app

```
$ npx cap open ios
$ npx cap open android
```
